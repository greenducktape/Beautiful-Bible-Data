# Contributing to Beautiful Bible Data

Thanks for your interest in contributing! Here's how to get involved.

## Reporting bugs

Open an [issue](https://github.com/greenducktape/Beautiful-Bible-Data/issues) and include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Browser and OS

## Suggesting features

Open an issue with the `enhancement` label. Describe the feature and why it would be useful.

## Submitting a pull request

1. Fork the repo and create a branch from `main`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
4. Make your changes
5. Open a pull request with a clear description of what you changed and why

## Code style

- TypeScript for all new files
- Tailwind CSS for styling — avoid inline styles
- Keep components focused; one responsibility per file
